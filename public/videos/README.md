# Hero video

Drop the hero background video here as:

    hero-cleaning.mp4

It is referenced by the homepage hero (`src/components/home/Hero.tsx`) at the
public path `/videos/hero-cleaning.mp4`.

Recommendations for clean performance:
- H.264 / MP4, 1080p, ~6–12s seamless loop, no audio track
- Keep the file lightweight (aim for < 5–8 MB) — it autoplays on every visit
- Pair it with the poster image at `/images/hero-poster.jpg`
